## Các bước cài đặt

1. Chạy file `HLK_CafeGiaiKhat_Deploy.bat` trong thư mục `Deploy` bằng quyền **Administrator** để thực hiện tự động tạo database, kết nối và sửa file `Web.config`.
2. Vào menu chọn **Build** > **Clean Solution** và chờ khoảng 02 giây.
3. Nhấn **F5** để chạy dự án.
